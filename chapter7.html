Chapter 7
Main verbs


7.1   Introduction

It is the main verb of a clause that determines which constituents (if any, besides the subject) are required to make the clause structurally complete. The annotation marks these requirements by adding to the word class tag of each main verb a code from the fourth edition of the Oxford Advanced Learner's Dictionary (OALD4; Cowie 1989). These verb codes:

(i)
partially establish word sense
(ii)
disambiguate the subject contribution for passivised verbs (notably, -DOB1 role with [Tnt] vs -OB2 role with [Dn.t])
(iii)
signal when adverbial elements are selected items of the clause
(iv)
provide a succinct notation to assist cataloging and searching clause structure
(v)
offer an independent check on the validity of the clause annotation

The chapter is organised as follows. Section 7.2 gives an overview of the OALD4 codes. Thereafter each code is described and illustrated, with section 7.3 covering linking verb codes; section 7.4, intransitive verb codes; section 7.5, monotransitive verb codes; section 7.6, complex-transitive verb codes; and section 7.7, ditransitive verb codes.


7.2   OALD4 verb codes

An OALD4 verb code starts with a capital letter: L, I, T, C, or D. This capital letter signals how the verb belongs to one of the verb classes of Table 7.1. These verb classes set requirements for the structural level of the clause: the NUMBER of elements and their FUNCTION. An element with subject function is always required in a basic clause, so mentioned elements are additional to the subject.

Table 7.1: Capital letters L, I, T, C, and D

L LINKING verb is associated with a SUBJECT PREDICATIVE (-PRD2), an element which provides information about the subject of the clause.
I INTRANSITIVE verb is NOT associated with a PREDICATIVE or an OBJECT, although it may be associated with an ADVERBIAL, an element which tells us about time, place, manner, etc of the action of the verb.
T TRANSITIVE verb MONO-TRANSITIVE verb is associated with a DIRECT OBJECT (-OB1), an element which often refers to the person or thing affected by the action of the verb.
C COMPLEX-TRANSITIVE verb is associated with both a DIRECT OBJECT (-OB1) and an OBJECT PREDICATIVE (-PRD), an element which provides more information about the direct object. Note: in the code, a dot divides information about the realisation of the direct object from information about the realisation of the object predicative.
D DITRANSITIVE verb is associated with both a DIRECT OBJECT (-OB1) and an INDIRECT OBJECT (-OB2), an element which refers to a person who receives something or benefits from an action. Note: in the code, a dot divides information about the realisation of the direct object from information about the realisation of the indirect object.

    A complete verb code is reached when a capital letter from Table 7.1 is possibly followed by lower case letters from Table 7.2.

Table 7.2: Lower case letters a, n, p, pr, n/pr, n/a, t, f, w, g, and i

aadjective phrase
nnoun phrase
padverb particle
prpreposition phrase
n/prnoun phrase/preposition phrase
n/aas + noun phrase/adjective phrase
tnon-finite clause (to-infinitive) (IP-INF with internal TO)
fthat-clause (CP-THT)
wfinite or non-finite clause with wh (CP-QUE)
gparticipial clause ({ing} form) (IP-PPL)
inon-finite clause (‘bare’ infinitive) (IP-INF without internal TO)

These lower case letters represent information about the FORM of the constituents that are used to realise the required elements that are signalled by their capital letter. For example, [La] is the code to mark a linking verb in clause structure with the linking verb + a subject predicative constituent that is an adjective phrase. In codes with multiple lower case letters, the lower case letters can appear separated by a dot (‘.’) character. For example, [Cn.a] is the code to mark a complex-transitive verb in clause structure with the complex-transitive verb + a direct object constituent that is a noun phrase + an object predicative constituent that is an adjective phrase.


7.3   Linking verb codes

This section introduces linking verb codes: [La] and [Ln].

7.5   Mono-transitive verb codes

This section introduces mono-transitive verb codes: [Tn], [Tn.pr], [Tn.p], [Tf], [Tw], [Tt], [Tnt], [Tg], [Tsg], [Tng], and [Tni].

7.5.1   [Tn]

[Tn] is the code to mark a monotransitive verb in clause structure with the monotransitive verb + a direct object that is a noun phrase.

  • The direct object is a noun phrase
  • For verbs used in this pattern and the [I] pattern without a change of subject or meaning see section 7.4.1.

    Some verbs used in this pattern can also be used on the [I] pattern only with what is the object of the [Tn] needing to become the subject of the [I] pattern. For example, compare (7.19) with (7.5) above.

(7.19)

"IP-MAT-OB1","IP-PPL-CAT","NP-OB1","NS","churches"
"ID","4#77_susanne_A10@16"
  • Most verbs in this pattern can be made passive, with the object of the active pattern becoming the subject of the passive one:
(7.20)

There are exception where no passive is possible [Tn no passive].

7.5.2   [Tn.pr]

[Tn.pr] is the code to mark a monotransitive verb in clause structure with the monotransitive verb + a direct object that is a noun phrase + an adverbial preposition phrase that is a selected item of the verb.

  • In this pattern, the verb is closely linked in grammar and meaning to a preposition phrase. The exact choice of preposition is shown in bold print in the above table.
  • After some verbs, the preposition phrase cannot be removed without changing the meaning of the verb or producing nonsense:
(7.21)
a.
The teacher referred the class to a passage in the textbook.
b.
*The teacher referred the class.
  • After other verbs, the preposition phrase can be removed without changing the meaning of the verb:
(7.22)
a.
The visiting speaker thanked the Chairman for his kind remarks. [Tn.pr]
b.
The visiting speaker thanked the Chairman. [Tn]
  • Most verbs in this pattern can be made passive. The direct object of an active verb becomes the subject of the same verb in the passive:
(7.23)
a.
The council have cleared the pavements of rubbish.
b.
The pavements have been cleared of rubbish (by the council).

There are exceptions where no passive is possible [Tn.pr no passive].

7.5.8   [Tg]

  • For remember and forget, see [Tt], note (b).

  • In this pattern, the direct object is a non-finite clause containing an {ing} form verb introduced EITHER by a noun phrase (e.g., him, her, us, John, the staff OR by a possessive form (e.g., his, her, our, John's, the staff's). 7.6   Complex-transitive verb codes

This section introduces Complex-transitive verb codes: [Cn.a], [Cn.n], [Cn.n/a], [Cn.pr], [Cn.t], [Cn.g], and [Cn.i].

7.6.1   [Cn.a]

[Cn.a] marks clause structure where there is a complex-transitive verb + direct object that is a noun phrase + object predicative that is an adjective phrase.

  • In this pattern, the object predicative is an adjective phrase which describes a feature or quality of the direct object.
  • Many verbs in this pattern can be made passive. The direct object of an active verb becomes the subject of the same verb in the passive:
(7.48)
a.
The CEGB made the situation slightly worse.
b.
The situation was made slightly worse (by the CEGB).

There are exceptions where no passive is possible [Cn.a no passive].

7.6.2   [Cn.n]

[Cn.n] is the code used to mark a complex-transitive verb in clause structure with the complex-transitive verb + a direct object that is a noun phrase + an object predicative that is a noun phrase.

  • In this pattern, the object predicative is a noun phrase which indicates the role, name, status, etc of the direct object.
  • Many verbs in this pattern can be made passive. The direct object of an active verb becomes the subject of the same verb in the passive:
(7.49)
a.
The court considered Smith a trustworthy witness.
b.
Smith was considered a trustworthy witness (by the court).

There are exceptions where no passive is possible [Cn.n no passive].

7.6.3   [Cn.n/a]

[Cn.n/a] is the code to mark a complex-transitive verb in clause structure with the complex-transitive verb + a direct object that is a noun phrase + an object predicative that is as + noun phrase/adjective phrase.

  • In this pattern, the object predicative tells us how the direct object is regarded, judged, etc, or what he, she, or it is chosen to act or serve as.

7.7   Ditransitive verb codes

This section introduces ditransitive verb codes: [Dn.n], [Dn.pr], [Dn.f], [Dpr.f], [Dn.w], [Dpr.w], and [Dn.t].

7.7.1   [Dn.n]

[Dn.n] is the code used to mark a ditransitive verb in clause structure with a ditransitive verb + an indirect object that is a noun phrase + a direct object that is a noun phrase.

  • This pattern has an indirect object (without a preposition) followed by a direct object. Both are noun phrases. The indirect but not the direct object can also be a personal pronoun.
  • When the indirect object refers to someone who receives something, this pattern can usually be changed to the [Dn.pr] pattern with to:
(7.59)
a.

b.
Some major organisations give additional compensation to staff. [Dn.pr]
  • When the indirect object refers to someone who is expected to benefit from the action of the verb, the [Dn.n] pattern can be changed to the [Dn.pr] with for:
(7.60)
a.
I'll make you a nice cup of tea. [Dn.n]
b.
I'll make a nice cup of tea for you. [Dn.pr]
  • For reasons why the [Dn.pr] pattern (with to or for) may be preferred to the [Dn.n] pattern, see the discussion of [Dn.pr].
  • Most verbs in this pattern can be made passive, with the indirect object becoming the subject:
(7.61)
a.
The Prime Minister has offered commonwealth governments every facility possible to safeguard their interests.
b.
Commonwealth governments have been offered (by the Prime Minister) every facility possible to safeguard their interests.

There are exceptions where no passive is possible [Dn.n no passive].
