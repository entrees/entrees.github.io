11.3.1   Tag questions

The annotation uses PRN in the description of tag questions. To find tag questions, you can use pattern (11.15):

(11.15)
/^PRN/ > /^IP-MAT/ < (/^CP-QUE/ < (/^IP-SUB/ <-1 /^NP-SBJ/))

This looks for a PRN node that is itself immediately under an IP-MAT, and that has immediately under it a CP-QUE. Furthermore the CP-QUE needs to immediately dominate an IP-SUB that has as its last child an NP-SBJ. Thus, (11.15) returns example sentences like (11.16).

(11.16)
( (IP-MAT (NP-SBJ (NPR Sirena))
          (BED;~Ln was)
          (NP-PRD2 (D a)
                   (N marvel))
          (PUNC ,)
          (__PRN (___CP-QUE (IP-SUB (BED was)

                                    (NP-SBJ (PRO she)))))
